# SnakeAI
I made an AI using Reinforcement Learning to play the classic Snake game.
It uses Deep Q Learning to train neural networks with the help of Pytorch framework.

## Libraries Used
-Pytorch
-Pygame
-numpy

## Record Score
After 8190 games,
Best Score: 86, Average Score: 32.39
